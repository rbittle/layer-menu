#Layer Menu

##Installation
Clone with:
``` bash
git clone https://github.com/rbittle/layer-menu.git
```

##Production
To download required packages, run:
``` bash
npm install --only=production
```
To build the project, run:
``` bash
npm run build
```
Run the server with:
``` bash
npm run server
```

##Development
To download required packages, run:
``` bash
npm install
```
To build the project, run:
``` bash
gulp
```
Run the dev server with:
``` bash
node .
```
